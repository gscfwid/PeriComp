## Project Description

This repository contains the code for the research paper "Enhancing Local Language Models for Perioperative Complication Detection: A Targeted Strategy with LoRa Fine-tuning".

The study explores how combining targeted prompt engineering with LoRa fine-tuning can boost performance of open-source models for perioperative complication detection from clinical records.
